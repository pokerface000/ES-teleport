# ES-teleport V1
Teleport For ESX server



# Telepítés
1. Telepítsd a scriptet a resources mappádba
2. server.cfg --> ensure ES-teleport
3. Indítsd újra a szervered

# Szükséges scriptek
[k5_notify](https://github.com/kac5a/k5_notify/)
